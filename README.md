# Accés a Dades

Aquest repositori conté el les practiques de l'assignatura d'**Accés a Dades**, fetes utilitzant Java.

## Llistat de practiques
  1. Consulta, creacio i modificacio de fitxers.
  2. Lectura i escriptura de fitxers.
